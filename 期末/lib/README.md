# 台州旅游 Flutter App

# 타이저우 여행 Flutter 앱

---

## 项目介绍

## 프로젝트 소개

这是一个基于 Flutter 开发的台州旅游展示应用。
该应用展示了台州著名景点、特色美食以及天气信息，并支持搜索与详情页面浏览。

이 프로젝트는 Flutter 기반으로 제작된 타이저우 관광 소개 앱입니다.
타이저우의 유명 관광지, 지역 음식, 날씨 정보를 제공하며 검색 및 상세 페이지 기능을 지원합니다.

---

## 功能特点

## 주요 기능

### 1. 景点展示

### 1. 관광지 소개

* 神仙居
* 台州府城
* 长屿硐天
* 路桥十里长街
* 椒江老街
* 温岭千年曙光园
* 大陈岛
* 天台山

각 관광지는 이미지,简介、评分以及详情页面。

각 관광지는 이미지, 설명, 평점 및 상세 페이지를 제공합니다.

---

### 2. 美食模块

### 2. 음식 소개

* 嵌糕
* 姜汤面
* 海鲜砂锅
* 食饼筒
* 麦虾
* 蛋清羊尾

支持美食详情页面展示。

음식 상세 페이지를 지원합니다.

---

### 3. 天气模块

### 3. 날씨 기능

* 今日天气展示
* 温度显示
* 天气状态显示

오늘의 날씨, 온도 및 날씨 상태를 제공합니다.

---

### 4. 搜索功能

### 4. 검색 기능

用户可以搜索台州景点。

사용자는 타이저우 관광지를 검색할 수 있습니다.

## 运行方法
