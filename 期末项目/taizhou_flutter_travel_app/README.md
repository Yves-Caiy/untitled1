# 台州家乡旅行推荐 App

这是一个符合期末项目要求的 Flutter 应用示例，主题为“台州，山海水城旅行推荐”。

## 已包含内容

- 家乡地区介绍：台州位置、山海水城、古城与海岸特色
- 代表旅游景点：神仙居、临海古城墙、温岭石塘
- 代表美食：台州食饼筒、台州麦虾、温岭嵌糕、东海海鲜
- 推荐旅行路线：台州一日游路线
- 图片及页面构成：底部导航、卡片、图标、山海主题色
- AI 使用内容：文案整理、页面设计、路线规划说明
- 项目执行心得：放在 AI 说明页面底部

## 运行方法

```bash
flutter pub get
flutter run
```

## 生成 APK

```bash
flutter build apk --release
```

生成位置：

```text
build/app/outputs/flutter-apk/app-release.apk
```

## 发表说明

大家好，我负责的是台州家乡旅行推荐页面。这个页面主要介绍台州的山海水城特色、代表景点、代表美食和一日游路线。我还加入了 AI 使用说明，说明 AI 帮助我整理文案、设计页面风格和规划旅行路线，最后由我进行检查和修改。
