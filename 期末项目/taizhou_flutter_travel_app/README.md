台州家乡旅行推荐 App / 타저우 고향 여행 추천 앱
 
这是一个符合期末项目要求的 Flutter 应用示例，主题为“台州，山海水城旅行推荐”。
기말 프로젝트 요구사항에 부합하는 Flutter 애플리케이션 예시로, 테마는 「산해수성 타저우 여행 추천」입니다.
 
已包含内容 / 포함된 콘텐츠
 
- 家乡地区介绍：台州位置、山海水城、古城与海岸特色
고향 지역 소개：타저우 지리적 위치, 산해수성 특징, 고성과 해안 특색
- 代表旅游景点：神仙居、临海古城墙、温岭石塘
대표 관광 명소：신선거, 임해 고성벽, 원링 스탄탕
- 代表美食：台州食饼筒、台州麦虾、温岭嵌糕、东海海鲜
대표 미식：타저우 식병통, 타저우 맥샤, 원링 감고, 동해 해산물
- 推荐旅行路线：台州一日游路线
추천 여행 코스：타저우 당일 여행 코스
- 图片及页面构成：底部导航、卡片、图标、山海主题色
이미지 및 페이지 구성：하단 네비게이션, 카드 레이아웃, 아이콘, 산해 테마 색상
- AI 使用内容：文案整理、页面设计、路线规划说明
AI 활용 내역：문안 정리, 페이지 스타일 디자인, 여행 코스 기획 지원
- 项目执行心得：放在 AI 说明页面底部
프로젝트 제작 소감：AI 설명 페이지 하단에 배치
 
运行方法 / 실행 방식
 
bash
  
flutter pub get
flutter run
 
 
生成 APK / APK 빌드 명령
 
bash
  
flutter build apk --release
 
 
生成位置 / 생성 경로：
 
text
  
build/app/outputs/flutter-apk/app-release.apk
